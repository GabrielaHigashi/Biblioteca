# LibraControl 

O **LibraControl** é um sistema de gestão bibliotecária desenvolvido como projeto final de curso. A aplicação permite o controle total de um acervo, incluindo o cadastro de livros, gestão de empréstimos, reservas e monitoramento de disponibilidade em tempo real através de um dashboard intuitivo.

## Tecnologias Utilizadas

* **Linguagem:** Java 17
* **Framework:** Spring Boot (Web, Data MongoDB, DevTools)
* **Banco de Dados:** MongoDB (Atlas)
* **Frontend:** HTML5, CSS3 e Thymeleaf 
* **Gerenciador de Dependências:** Maven

## Como implementar o projeto localmente

Siga os passos abaixo para configurar o ambiente e rodar a aplicação em sua máquina:

### 1. Pré-requisitos
Certifique-se de ter instalado:
* [JDK 17](https://www.oracle.com/java/technologies/downloads/#java17) ou superior.
* [Maven](https://maven.apache.org/download.cgi) (ou use o Maven Wrapper incluso no projeto).
* Uma conta no [MongoDB Atlas](https://www.mongodb.com/cloud/atlas).
